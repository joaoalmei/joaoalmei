## Main Projects

### <img src="https://github.com/user-attachments/assets/2f1e46d5-5472-4558-b75f-7f74600cac93" height="20"/> [The Office](https://github.com/joaoalmei/pmv-ads-2024-2-e4-proj-infra-t5-the-office-1)
